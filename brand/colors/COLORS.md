# Cores — PJCore Labs

## Paleta principal

| Uso | Cor |
|---|---|
| Fundo principal | `#0f1117` |
| Fundo secundário | `#141823` |
| Painel | `#171d29` |
| Painel suave | `#1d2433` |
| Texto principal | `#ffffff` |
| Texto secundário | `#aeb7c2` |
| Verde/neon principal | `#00ff88` |
| Cyan tecnológico | `#00d2ff` |
| Bordas suaves | `rgba(255, 255, 255, 0.09)` |

## Regra visual

A marca deve manter base escura, texto claro e destaques em verde/neon ou cyan tecnológico.

O verde/neon passa a ser a cor emocional da marca.  
O cyan pode continuar como cor secundária de interface.
