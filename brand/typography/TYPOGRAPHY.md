# Tipografia — PJCore Labs

## Fonte do website

O website usa fontes do sistema:

```css
system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
```

## Estilo

A tipografia deve manter:

- títulos fortes;
- letras limpas;
- boa leitura;
- espaçamento generoso;
- aparência tecnológica sem perder clareza.

## Uso

- Títulos principais: grandes, fortes, com impacto.
- Texto normal: claro, simples e legível.
- Eyebrows/badges: maiúsculas, espaçamento largo, cor de destaque.
