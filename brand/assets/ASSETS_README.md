# PJCore Labs — Brand Assets Draft

Versões geradas a partir da direção visual escolhida: fundo dark sci-fi, metálico, verde neon e slogan `IDEAS INTO REALITY`.

## Uso recomendado

- `pjcorelabs-logo-main-poster.png` — poster/base visual principal.
- `pjcorelabs-wallpaper-1920x1080.png` — wallpaper desktop Full HD.
- `pjcorelabs-wallpaper-2560x1440.png` — wallpaper desktop QHD.
- `pjcorelabs-og-image-1200x630.png` — imagem social/Open Graph para website.
- `pjcorelabs-symbol-no-inner-text.png` — símbolo limpo para favicon/app icon e usos pequenos.
- `pjcorelabs-symbol-with-inner-slogan.png` — símbolo com slogan dentro da circunferência para versões compactas onde fizer sentido.
- `favicon.ico` e `favicon-*.png` — favicons.

## Regra visual definida

Logo principal/wallpaper:
- circunferência e triângulo sem texto dentro;
- wordmark `PJCORE LABS` por baixo;
- slogan `IDEAS INTO REALITY` fora da circunferência.

Logo pequeno:
- pode usar o slogan dentro da circunferência;
- para favicon muito pequeno, usar preferencialmente a versão sem texto para manter legibilidade.
