Project

Workstation OS

A personal workstation control system for monitoring, maintenance, productivity and quick access to local projects.

View repository Back to projects

Purpose

A control centre for the workstation.

Workstation OS is designed to centralise the daily management of a personal Linux workstation: system status, maintenance actions, project shortcuts, GitHub access, productivity tools and useful diagnostics.

Status

In active development.

Focus

Practical workstation management, not a full operating system replacement.

Platform

Linux-first, browser-based dashboard with local tooling.

Direction

Simple interface, clear diagnostics and fast access to important actions.

Core features

What it is being built to do.

System status

CPU, RAM, disk, temperature, internet and general machine state in one place.

Maintenance

Quick access to useful scripts, updates, checks and future repair/diagnostic actions.

Project launcher

Fast access to active local projects, development folders, GitHub repositories and tools.

Plain-language diagnostics

Technical information presented in a way that is useful without being buried in terminal output.

Roadmap

Next direction.

  1. Stabilise the dashboard structure and core system cards.
  2. Improve diagnostics and make technical status easier to understand.
  3. Add safer maintenance actions with clear confirmations.
  4. Connect project documentation, scripts and GitHub workflow.
  5. Prepare a cleaner product identity under PJCore Labs.
